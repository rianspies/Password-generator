Project Overview

The Random Password Generator is a web-based application that generates secure, random passwords using a mix of characters. The project includes an intuitive user interface with a password display field, a copy-to-clipboard feature, and real-time notifications to enhance usability.

This project is designed as an intermediate to advanced learning exercise, focusing on practical JavaScript functionality and user interaction.

Features

* Generate a random password at the click of a button
* Uses a mix of characters (letters, numbers, and symbols)
* One-click copy-to-clipboard functionality
* Visual notification confirming successful copy
* Clean and user-friendly interface

---

Skills & Concepts Covered

* Random password generation logic
* JavaScript string manipulation
* Clipboard API (copy text programmatically)
* Event handling
* UI feedback using notifications
* Building reusable and interactive components

---

How It Works

1. Click the Generate button.
2. A random password is created and displayed in the input field.
3. Click the copy icon to copy the password.
4. A notification appears confirming the password has been copied.
5. Paste the password anywhere you need.
